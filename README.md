# Blackjack 21

Projeto desenvolvido para a disciplina de Estrutura de Dados II.

## Objetivo
Demonstrar a aplicação da estrutura de dados Pilha (Stack) através da implementação do jogo Blackjack (21).

## Tecnologias
- Java
- JSP
- Servlets
- HTML/CSS

## Estrutura de Dados Utilizada
- Pilha (Stack)

## Funcionalidades
- Iniciar partida
- Comprar carta
- Parar jogada
- Cálculo automático de pontuação
- Verificação de vitória, derrota ou empate
